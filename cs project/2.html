Creating a project

The Design Process

When you code something, you are solving a problem, be it client requests, a convenience tool, or just plain boredom. When you are solving a problem, you will need a solution. Be sure to know who will use your code and what your code is supposed to do. Be creative in finding ways to address certain problems with quality.

Analyzing the problem

It helps to break down a problem into multiple, smaller and easeir ones. Ideally, you should also know how to use functions so as to have a solution for a recurring problem in your code.

Photo by Kaleidico on Unsplash

Attention to Detail

Although you don't need to be too strict with your code, it helps to be observant in the event you forget something trivial. A missing character can be all it takes for the entire project to break.

Photo by Agence Olloweb on Unsplash

Organization

It helps to keep organized. Use the tab key to indent lines that are inside certain functions. There are also some coding conventions that should be helpful to follow.
