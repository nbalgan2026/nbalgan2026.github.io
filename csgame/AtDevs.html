Cheating :)

count is your currency. Change it, and you change how much currency you have


count = 1000; will set your currency to 1000


The functions to buy and trigger upgrades will check if you have the currency to buy them, so simply typing the functions in the console manually will not work

Instead, set count to a very big number to buy and test an upgrade
